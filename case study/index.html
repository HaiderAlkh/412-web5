Background

This project was for web 4 last semester and the goal was to create a website in english and french for an amusement park and all the information that goes with it. So my group chose this Finnish amusment park called Linnanmäki Amusement Park, the design should showcase the vibe of the park and the rides that are there. The website should also be easy to navigate and have a good flow to it. The first problem for us was the color palette and the layout of the site.


Process

We first decided what type of site should be and easily chose it as a single page app, since clients could easily see all the necessary information fairly quickly. We then decided on the color palette and the layout, we wanted simple colors that catch peoples attention and was youthful, so, orange and a grid layout.



Results

The outcome is that we passed and got a good grade, the site is easy to navigate and has a good flow to it. The site is also responsive and works on mobile devices. The site is also in english and french, so it can be used by more people.
